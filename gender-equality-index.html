Gender Equality Index

Motivated by the International Working Women's Day, I wanted to explore the Gender Equity Index dataset released by SocialWatch with their indicators of Gender Quality Index. In particular I wanted to find out how is my own country, Chile, in terms of equality in comparison with other countries.

This was a experiment to learn how to work with maps in d3.js. See the visualization here.

Everyone should have the same access to the same opportunities. Hopefully, one day these maps will have only one color.
